PLACE ODA FILE CONVERTER HERE
================================

For DWG file support, download and place ODA File Converter in this folder:

Windows:
  - Download: https://www.opendesign.com/guestfiles/oda_file_converter
  - Place: ODAFileConverter.exe (in this folder)

Linux/Mac:
  - Download: https://www.opendesign.com/guestfiles/oda_file_converter
  - Place: ODAFileConverter (in this folder)
  - Run: chmod +x ODAFileConverter

After setup, server will automatically convert DWG files to DXF.

See SETUP_DWG.md in project root for detailed instructions.
